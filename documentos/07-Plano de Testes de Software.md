# Plano de Testes de Software

Os testes funcionais a serem realizados na aplicação são descritos a seguir.

# 📋 Plano de Testes de Software

| **Caso de Teste** | **Requisitos Associados** | **Objetivo do Teste** | **Passos** | **Critérios de Êxito** | **Responsável** |
|-------------------|----------------------------|------------------------|------------|-------------------------|-----------------|
| **CT-01: Verificar funcionamento da página de login** | RF-02/07 | Verificar se a página de login permite acesso com usuário e senha. | 1. Acessar o site.<br>2. Clicar em "Login".<br>3. Inserir usuário e senha.<br>4. Clicar em "Entrar". | O sistema deve autenticar o usuário e redirecionar para seu painel. | Welinton |
| **CT-02: Verificar cards de vagas e envio de currículo** | RF-02 | Verificar se os cards das vagas estão visíveis e se é possível se candidatar e enviar currículo. | 1. Acessar o site.<br>2. Fazer login como candidato.<br>3. Navegar até a página de vagas.<br>4. Verificar os cards.<br>5. Clicar em "Candidatar-se" e enviar currículo. | O card deve exibir os dados da vaga e permitir envio do currículo com confirmação de sucesso. |Edilson |
| **CT-03: Verificar barra de filtragem de vagas** | RF-06 | Verificar se o filtro de vagas funciona corretamente. | 1. Fazer login como candidato.<br>2. Acessar a página de vagas.<br>3. Utilizar os filtros (por cidade, bairro, área, etc). | O sistema deve exibir apenas as vagas que correspondem aos filtros selecionados. | Pedro |
| **CT-04: Verificar cadastro de vagas pela empresa** | RF-08 | Validar se a empresa consegue cadastrar novas vagas. | 1. Fazer login como empresa.<br>2. Acessar "Cadastrar Vaga".<br>3. Preencher todos os campos.<br>4. Clicar em "Salvar". | A vaga deve ser salva e exibida na listagem de vagas com confirmação de sucesso. | Pedro |
| **CT-05: Verificar filtros por área de interesse** | RF-10 | Verificar se o sistema filtra corretamente as vagas por cidade, bairro e área. | 1. Fazer login como candidato.<br>2. Acessar a página de vagas.<br>3. Utilizar os filtros de cidade, bairro e área de interesse.<br>4. Verificar os resultados. | O sistema deve retornar apenas vagas que se enquadram nos filtros aplicados. | Pedro |
| **CT-06: Verificar seção de candidatos inscritos para empresas** | RF-07 | Validar se a empresa consegue visualizar os candidatos inscritos nas vagas. | 1. Fazer login como empresa.<br>2. Acessar a página "Minhas Vagas".<br>3. Clicar na vaga desejada.<br>4. Verificar a lista de candidatos. | Deve exibir corretamente todos os candidatos inscritos na vaga. | Gabriel |
| **CT-07: Verificar envio de arquivos pela plataforma** | RF-02 | Verificar se o envio de arquivos (como currículos) funciona corretamente. | 1. Fazer login como candidato.<br>2. Acessar uma vaga.<br>3. Clicar em "Candidatar-se".<br>4. Anexar o arquivo.<br>5. Clicar em "Enviar". | O sistema deve aceitar o arquivo, realizar upload e confirmar o envio |  Ana |
| **CT-08: Verificar geração de dashboard com gráficos** | RF-11 | Verificar se o sistema gera corretamente o dashboard com gráficos de vagas e candidatos.| 1. Acesse a seção de dashboard no menu. 2. Verifique se os gráficos estão visíveis.  3. Confirme se os dados exibidos correspondem às vagas e candidatos registrados no sistema. | O sistema deve gerar os gráficos correspondentes as vagas e candidatos | Edilson |
